# PhysicsEngine.rad2deg()

## Overview

Converts radians to degrees.

## Syntax

```javascript
const degrees = PhysicsEngine.rad2deg(radians);
```

## Parameters

- `radians` (number): Angle in radians

## Returns

- (number): Angle in degrees

## Example

```javascript
const deg = PhysicsEngine.rad2deg(Math.PI);  // 180
```
