# PhysicsEngine.deg2rad()

## Overview

Converts degrees to radians for angular calculations.

## Syntax

```javascript
const radians = PhysicsEngine.deg2rad(degrees);
```

## Parameters

- `degrees` (number): Angle in degrees

## Returns

- (number): Angle in radians

## Example

```javascript
const rad = PhysicsEngine.deg2rad(180);  // 3.14159...
```
