# PhysicsEngine Constants

## Overview

The `PhysicsEngine` module provides fundamental constants for orbital mechanics calculations.

## Constants

| Constant | Value | Description |
|----------|-------|-------------|
| G | 6.67430e-11 | Gravitational constant (m³ kg⁻¹ s⁻²) |
| AU | 149597870700 | Astronomical Unit (meters) |
| DAY | 86400 | Seconds per day |
| DEG2RAD | π/180 | Degrees to radians |
| RAD2DEG | 180/π | Radians to degrees |

## Usage

```javascript
const G = PhysicsEngine.G;
const au = PhysicsEngine.AU;
```
