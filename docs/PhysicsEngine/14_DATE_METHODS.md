# PhysicsEngine Date Methods

## getFormattedDate()

Returns formatted date string.

```javascript
const dateStr = PhysicsEngine.getFormattedDate();  // "YYYY-MM-DD"
```

## getJulianDate()

Calculates Julian Date.

```javascript
const jd = PhysicsEngine.getJulianDate();  // e.g., 2451545.0
```

## getAllPositions()

Returns positions of all bodies.

```javascript
const positions = PhysicsEngine.getAllPositions();
```
