# PhysicsEngine State

## State Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| simTime | number | 0 | Days since J2000 |
| timeScale | number | 1 | Time multiplier |
| isPlaying | boolean | false | Playback state |

## Usage

```javascript
PhysicsEngine.simTime;     // Current simulation time
PhysicsEngine.timeScale;   // Time speed multiplier
PhysicsEngine.isPlaying;   // Is simulation running
```
