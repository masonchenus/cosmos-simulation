# PhysicsEngine Time Control Methods

## setTime()

Sets simulation time.

```javascript
PhysicsEngine.setTime(time);  // time: days since J2000
```

## update()

Advances simulation time.

```javascript
PhysicsEngine.update(deltaTime);  // deltaTime: seconds
```

## setTimeScale()

Sets time multiplier.

```javascript
PhysicsEngine.setTimeScale(scale);  // 1 = real time
```

## play(), pause(), reset()

Control simulation state.
