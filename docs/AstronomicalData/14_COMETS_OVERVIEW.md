# AstronomicalData Comets Overview

## Overview

The `Comets` object contains data for notable comets in the solar system. Comets have highly elliptical orbits with high eccentricity.

## Comet List

- Halley's Comet
- Comet Hale-Bopp
- Comet Encke
- Comet Swift-Tuttle
- Comet Tempel-Tuttle

## Accessing Comet Data

```javascript
const comets = AstronomicalData.Comets;
const halley = AstronomicalData.Comets.Halley;
```
