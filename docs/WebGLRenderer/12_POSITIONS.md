# WebGLRenderer Position Methods

## updatePositions(time)

Updates celestial body positions.

```javascript
WebGLRenderer.updatePositions(TimeSystem.getDaysSinceJ2000());
```

## updateOrbits(time)

Updates orbital path calculations.

```javascript
WebGLRenderer.updateOrbits(time);
```

## toggleOrbits(visible)

Toggles orbit line visibility.

```javascript
WebGLRenderer.toggleOrbits(true);
```
