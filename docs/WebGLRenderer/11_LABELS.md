# WebGLRenderer Label Methods

## createLabel(name, offset)

Creates HTML label for a celestial body.

```javascript
WebGLRenderer.createLabel('Mars', {x: 0, y: 1, z: 0});
```

## updateLabels()

Updates label positions based on camera projection.

```javascript
WebGLRenderer.updateLabels();
```

## toggleLabels(visible)

Toggles label visibility.

```javascript
WebGLRenderer.toggleLabels(true);
```
