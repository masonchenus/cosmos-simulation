# WebGLRenderer.createSun()

## Overview

Creates the Sun mesh with glow and corona effects.

## Syntax

```javascript
WebGLRenderer.createSun();
```

## Visual Elements

- Sun sphere (radius: 5 units)
- Inner glow (radius: 6 units, opacity: 0.3)
- Corona (radius: 8 units, opacity: 0.1)
- Label at position {x: 0, y: 8, z: 0}
