# TimeSystem Constants

## Overview

The `TimeSystem` module provides constants for time calculations.

## Constants

| Constant    | Value       | Description             |
| ----------- | ----------- | ----------------------- |
| DAY         | 86400000    | Milliseconds per day    |
| HOUR        | 3600000     | Milliseconds per hour   |
| MINUTE      | 60000       | Milliseconds per minute |
| SECOND      | 1000        | Milliseconds per second |
| J2000_EPOCH | Date object | J2000 epoch reference   |
| J2000_JD    | 2451545.0   | Julian Date of J2000    |

## Usage

```javascript
const dayMs = TimeSystem.DAY;  // 86400000
```
