# TimeSystem Jump Methods

## goToDate(year, month, day)

Jumps to specific calendar date.

```javascript
TimeSystem.goToDate(1969, 7, 20);  // Moon landing
```

## jumpForward(amount, unit)

Advances time by specified amount.

```javascript
TimeSystem.jumpForward(1, 'weeks');
```

## jumpBackward(amount, unit)

Rewinds time by specified amount.

```javascript
TimeSystem.jumpBackward(1, 'months');
```
