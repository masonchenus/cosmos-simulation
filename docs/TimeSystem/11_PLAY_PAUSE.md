# TimeSystem Play/Pause Methods

## play()

Starts/resumes simulation.

```javascript
TimeSystem.play();
```

## pause()

Pauses simulation.

```javascript
TimeSystem.pause();
```

## togglePlay()

Toggles between play and pause.

```javascript
const isPlaying = TimeSystem.togglePlay();
```

## reset()

Resets simulation to initial time.

```javascript
TimeSystem.reset();
```
