# TimeSystem Format Methods

## formatDate()

```javascript
const str = TimeSystem.formatDate();  // "YYYY-MM-DD"
```

## formatDateTime()

```javascript
const str = TimeSystem.formatDateTime();  // "YYYY-MM-DD HH:MM:SS"
```

## formatRelativeTime()

```javascript
const str = TimeSystem.formatRelativeTime();  // e.g., "1 years"
```
